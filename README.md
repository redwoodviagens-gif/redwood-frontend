# Redwood Viagens — Frontend Next.js

Frontend profissional para consumir o backend Railway da Redwood.

## Rodar localmente

```bash
npm install
cp .env.example .env.local
npm run dev
```

Abra: http://localhost:3000

## Variáveis

```env
NEXT_PUBLIC_API_URL=https://redwood-backend-production.up.railway.app
NEXT_PUBLIC_WHATSAPP_PHONE=5555992290849
```

## Deploy

Pode hospedar na Vercel ou exportar/adaptar para Hostinger. Para Node/Next completo, Vercel é o caminho mais simples.
